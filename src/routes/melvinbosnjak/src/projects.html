My Projects

Asteroid Belt

You are the pilot of the Millennium Falcon and you're approaching an asteroid belt. You can use the left and right arrow keys to avoid the oncoming asteroids and make it as far as you can. The score is incremented for each asteroid that you successfully avoid. I wrote this game in plain Javascript drawing to a canvas element.

Space Wars

Space Wars is a multiplayer game that allows many clients to connect at one time. The goal of the game is to eliminate other players and avoid hitting the star in the center. The star has a gravitational pull making the game a little more difficult and interesting. Both the client and the server for this project were programmed in C# using Visual Studio. The network controller makes use of sockets to communicate between client and server. Scores and stats were saved to an SQL database using mySQL.

Spreadsheet

This application implements the functionality of a spreadsheet similar to Excel. It allows users to store strings, numbers, or formulas into each cell. The use of formulas are demonstrated in the image above where the value of the B1 cell is dependent on the values of the A1 and A2. The spreadsheet also allows multiple users to connect to a spreadsheet at one time so that when one client makes a change to the spreadsheet server, then the change is propogated to all other clients connected to that server. All of the libraries and the GUI were developed in C# using Visual Studio.

FPGA Tank Game

Worked on a team of 4 to design a single core CPU with a suite of custom instructions inspired by the RISC architecture. Using our custom assembly language, we wrote a top down tank shooter game that supports multiple tank and projectile sprites and used SNES controllers to control the tanks. Additional finite state machines handled the SNES input and the VGA display for the game. The project was written in Verilog and embedded onto the Terasic DE1-SoC FPGA board.

Speaker Circuit

Calculated, Simulated, Prototyped and Measured a Microphone-Speaker circuit. Designed with a custom BJT Amplifier with a gain of 120, and a dual rail, any voltage, regulated, DC power supply capable of supplying enough current and voltage to operate the microphone and speaker circuit. Simulations using PSPICE.

Poker using stochastic sampling

Created a program that computes the odds of getting any specific type of poker hand (based on 5 card or 7 card draws). These were determined with uses of exhaustive comparison and stochastic sampling using a Random number generator. Added a Texas hold 'em feature that stochastically computes the likely-hood a set of two cards will beat another set of two cards. Link to most of my projects is provided.
