My Projects

Space Wars

Space Wars is a multiplayer game that allows many clients to connect at one time. The goal of the game is to eliminate other players and avoid hitting the star in the center. The star has a gravitational pull making the game a little more difficult and interesting. Both the client and the server for this project were programmed in C# using Visual Studio. The network controller makes use of sockets to communicate between client and server. Scores and stats were saved to an SQL database using mySQL.

Universal Asynchronous Receiver/Transmitter

This circuit can communicate with a standard computer terminal. It is able to accept serial ASCII data from a terminal keyboard (when you press a key on the terminal, serial ASCII data is sent out the serial connection) and collect it into a parallel byte of information. This byte of data will then be sent back to the terminal as serial data so that it is echoed on the terminal screen. Written in Verilog using finite state machines, and embedded to a Nexys3 board using Adept.

Speaker Circuit

Calculated, Simulated, Prototyped and Measured a Microphone-Speaker circuit. Designed with a custom BJT Amplifier with a gain of 120, and a dual rail, any voltage, regulated, DC power supply capable of supplying enough current and voltage to operate the microphone and speaker circuit. Simulations using PSPICE.

Poker using stochastic sampling

Created a program that computes the odds of getting any specific type of poker hand (based on 5 card or 7 card draws). These were determined with uses of exhaustive comparison and stochastic sampling using a Random number generator. Added a Texas hold 'em feature that stochastically computes the likely-hood a set of two cards will beat another set of two cards. Link to most of my projects is provided.
